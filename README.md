# airQualityMonitor_connect
Updating air quality level from MQ135 sensor to connect using Arduino UNO and a web app

# Connections:
For this project we used Arduno UNO and MQ135 gas sensor.

| Arduino UNO        | MQ135 |
| ------------------ | ----- |
| 5 V                | VCC   |
| GND                | GND   |
| A0                 | A0    |
| Pin2               | D0    |
